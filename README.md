## Эмулятор машины Тьюринга

### Описание проекта

**Эмулятор машины Тьюринга** — это графическое приложение для моделирования работы абстрактной машины Тьюринга. Программа позволяет создавать, настраивать и запускать алгоритмы на основе классической модели машины Тьюринга.

### Основные возможности

* Настройка алфавитов ленты и дополнительных символов
* Создание и редактирование программы переходов
* Визуализация работы машины в реальном времени
* Пошаговое выполнение программы
* Управление скоростью выполнения
* Возможность сброса состояния машины

### Структура проекта

Проект состоит из следующих основных компонентов:

* **MainWindow** — главное окно приложения с интерфейсом управления
* **TuringMachine** — класс, реализующий логику работы машины
* **TapeWidget** — виджет для визуализации ленты
* **SetupDialog** — диалоговое окно настройки алфавитов
* **CMakeLists.txt** — файл конфигурации сборки

### Требования

* Qt Framework (версия 5)
* CMake (версия 3.16 или выше)
* Компилятор C++17

### Использование

1. Запустите приложение
2. Настройте алфавиты в диалоговом окне
3. Создайте программу переходов
4. Задайте входное слово
5. Запустите выполнение
